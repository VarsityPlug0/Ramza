Background Images Folder
========================

This folder is for background images that will be displayed on menu, cart, and checkout pages.

Supported formats: .png, .jpg, .jpeg, .gif, .webp

How to add background images:
1. Add your image files to this folder
2. The system will randomly select one image for each page load
3. Images should be high resolution (1920x1080 or higher) for best results
4. The images will have an overlay applied for text readability

Example filenames:
- restaurant-bg-1.jpg
- food-background.png  
- dining-ambiance.jpeg

The background images will NOT appear on:
- Home page (/)
- Admin/Dashboard pages (/dashboard/)

They WILL appear on:
- Menu page (/menu/)
- Cart page (/cart/)
- Checkout page (/checkout/)
- Any other content pages
